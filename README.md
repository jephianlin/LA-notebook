# 線性代數筆記本

通常我們會拿筆記本來做一些計算、看看例子、或是拿來寫筆記。  
線性代數筆記本的概念也是一樣的；  
當你學到一個新的概念﹐你應該要**動手算一算**、**觀察一些例子**（正面的或反正的！）、最後把寫下**屬於你自己的筆記**。  
這比較像是一個做實驗的過程﹐實驗執行者是是自己的大腦﹐而實驗對象是各個數學概念；  
好好地把這份實驗紀錄下來﹐讓你未來的大腦可以快速回顧自己曾經做過的事吧！
